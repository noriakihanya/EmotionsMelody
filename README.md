# EmotionsMelody
***- Your words will be music. -***


日本語の感情分析ライブラリHuggingface Transformersを用いてあなたの歌詞を分析し、歌詞の感情に適したコード進行を提示する。提示されたコード進行に基づいて音楽生成ライブラリMagentaで自動で音楽を生成する。

# DEMO
